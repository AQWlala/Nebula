主要优化建议：

Nebula
在 Tauri IPC 场景中，错误可能来自三个层面：Rust 后端 panic、序列化失败、网络超时。建议定义统一的 Result<T, NebulaError> 类型：
enum NebulaError {
  Backend(code, msg),
  Timeout(ms),
  Serialize(field),
}
这样前端可以根据 error.kind 做针对性降级处理，而不是所有错误都弹一个 toast。
