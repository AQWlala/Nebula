# v0.5.0 — 写作/工作双模式与 E2EE 同步

> **发布日期**: 2025-11-01
> **类型**: 大版本 (Minor version bump)
> **前一版本**: v0.3.0
> **变更范围**: 模式系统, 编辑器, OS 集成, 同步

## 变更摘要
落地写作模式与工作模式双轨,引入内置编辑器(Monaco + xterm + Git 集成),完善 OS 集成(剪贴板 / 通知 / 托盘 / 快捷键),并实现端到端加密(E2EE)同步以支持多设备协同。

## 新增功能
1. 写作模式 / 工作模式 — 场景化 UI 与工具集切换
2. 内置编辑器 — Monaco(代码)+ xterm(终端)+ Git 集成
3. OS 集成 — 剪贴板 / 通知 / 托盘 / 快捷键 / 文件关联
4. E2EE 同步 — 端到端加密的多设备数据同步

## 修改的功能
1. 主界面: 引入模式切换器(ModeSwitcher)
2. 存储层: 新增同步与设备管理子模块
