# v2.3.0 — macOS 风格全前端重设计

> **发布日期**: 2026-07-11
> **类型**: 大版本 (Minor version bump)
> **前一版本**: v2.2.0
> **变更范围**: 三栏布局, 毛玻璃侧边栏, Spotlight 工具栏, 11 核心视图, 设置页面

## 变更摘要
macOS 风格全前端重设计:采用三栏布局,落地毛玻璃侧边栏与 Spotlight 工具栏,重设计 11 个核心视图,并将设置页面改造为 macOS 系统偏好风格。

## 新增功能
1. 三栏布局 — 左侧边栏 / 中间主内容 / 右侧辅助面板
2. 毛玻璃侧边栏 — backdrop-blur 半透明效果
3. Spotlight 工具栏 — 全局快速搜索工具栏
4. 11 个核心视图重设计 — Dashboard / Memory / Swarm / Workflow / Calendar / Settings 等
5. 设置页面 — macOS 系统偏好风格(分组列表 + 详情面板)

## 修改的功能
1. 全前端视觉系统 — macOS 设计语言统一
2. 布局结构 — 从单栏/双栏升级为三栏
