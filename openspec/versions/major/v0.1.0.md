# v0.1.0 — 首个 Release: Nebula 脚手架与核心引擎

> **发布日期**: 2025-04-01
> **类型**: 大版本 (Minor version bump)
> **前一版本**: -
> **变更范围**: 脚手架, 记忆子系统, 引擎, LLM gateway, Swarm, 存储

## 变更摘要
Nebula 首个 release,完成 Tauri+Preact 前端脚手架搭建,落地 8 层记忆子系统(L0-L7),实现 Sponge/Blackhole 双引擎,集成 LLM gateway(以 Ollama 为初始 Provider),搭建 Swarm 蜂群(coder/writer/reviewer 三角色),并使用 SQLite + LanceDB 作为存储底座。

## 新增功能
1. Tauri + Preact 前端脚手架(含基础窗口、IPC 通道、热重载开发流)
2. 8 层记忆子系统(L0 Cache → L7 Archive)骨架与数据模型
3. Sponge 引擎(吸收/聚合)与 Blackhole 引擎(压缩/归档)
4. LLM gateway 初版,集成 Ollama 作为本地 Provider
5. Swarm 蜂群初版,落地 coder / writer / reviewer 三个基础 Agent
6. SQLite(结构化)+ LanceDB(向量)双存储底座
