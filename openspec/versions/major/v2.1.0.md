# v2.1.0 — Phase 1 可见性 + Phase 2 生态

> **发布日期**: 2026-07-10
> **类型**: 大版本 (Minor version bump)
> **前一版本**: v2.0.0
> **变更范围**: 模型配置中心, 对话框, 向导, 技能生态, 自动发明, 模型健康, 热更新, 技能调试, OpenClaw

## 变更摘要
Phase 1 可见性提升 + Phase 2 生态增强。落地模型配置中心、对话框宽度可配、首次启动向导、技能生态(市场/导入导出/调试)、技能自动发明、模型健康面板、模型自动发现、模型热更新、技能调试面板与 OpenClaw 集成。

## 新增功能
1. 模型配置中心 — 统一管理多 Provider / Model,支持增删改查与健康检测
2. 对话框宽度可配 — 用户自定义聊天面板宽度
3. 首次启动向导 — 引导新用户完成初始配置
4. 技能生态 — 技能市场 / 技能导入导出 / 技能调试面板
5. 技能自动发明 — AutoInventor 基于任务模式自动生成候选技能
6. 模型健康面板 — 实时展示 Provider 延迟 / 错误率 / 可用性
7. 模型自动发现 — 自动扫描本地 Ollama 可用模型
8. 模型热更新 — 运行时切换模型无需重启
9. 技能调试面板 — 单步执行 / 断点 / 变量查看
10. OpenClaw 集成 — 外部工具调用框架接入

## 修改的功能
1. 技能系统 — 从 CRUD 升级为完整生态
2. LLM gateway — 多 Provider 统一调度与健康监测
