# v0.2.0 — L5 Reflection 引擎与多粒度摘要

> **发布日期**: 2025-06-20
> **类型**: 大版本 (Minor version bump)
> **前一版本**: v0.1.0
> **变更范围**: L5 反思层, 后台 worker, Blackhole 压缩, 摘要引擎

## 变更摘要
引入 L5 Reflection 引擎与后台 worker 机制,使记忆系统具备自我反思与异步整理能力;完善 Blackhole 压缩管线,落地多粒度摘要(short / mid / long),为后续知识图谱与 MOC 自动整理奠定基础。

## 新增功能
1. L5 Reflection engine — 周期性反思记忆,产出 self-reflection 记录
2. 后台 worker — 独立于主线程执行反思 / 压缩 / 摘要任务
3. Blackhole 压缩管线完善 — 分级压缩与归档策略
4. Multi-granularity summary — 多粒度摘要(short / mid / long)

## 修改的功能
1. 记忆层架构: L5 由"预览"升级为完整实现
2. Blackhole 引擎: 压缩策略细化,接入 worker 调度
