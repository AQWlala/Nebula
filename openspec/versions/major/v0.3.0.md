# v0.3.0 — gRPC 服务化与 Skill CRUD

> **发布日期**: 2025-08-15
> **类型**: 大版本 (Minor version bump)
> **前一版本**: v0.2.0
> **变更范围**: gRPC 服务, Skill 系统, Memory read-side

## 变更摘要
引入 gRPC 服务层(基于 tonic 0.12,定义 22 个 RPCs),为外部客户端与 sidecar 提供高性能通道;落地 Skill CRUD 全流程,使技能可被创建/查询/更新/删除;补齐 Memory read-side 命令,支持从外部按层 / 类型 / 时间检索记忆。

## 新增功能
1. gRPC 服务(tonic 0.12, 22 RPCs)— 覆盖 memory / skill / swarm / reflect 等域
2. Skill CRUD — 技能全生命周期管理(创建 / 读取 / 更新 / 删除)
3. Memory read-side commands — 按层 / 类型 / 时间 / 标签检索记忆

## 修改的功能
1. 服务端架构: 在 REST 之外新增 gRPC 通道
2. Skill 系统: 从只读升级为可写

## 破坏性变更
(none — gRPC 为新增通道,REST 保持兼容)
